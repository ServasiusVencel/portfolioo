About me

Halo, nama saya Servasius Vencel, teman - teman biasa memanggil saya vencel. Saya siswa kelas 12 SMK Wikrama Bogor mengambil jurusan Pengembangan Perangkat Lunak dan Game (PPLG). Banyak hal yang saya pelajari selama di SMK Wikrama, mulai dari bahasa pemrograman, database sampai pembentukan karakter yang juga tidak kalah penting membentuk saya menjadi lebih siap dan matang di dunia kerja. Saya fokus ke sistem pengembangan backend yang berhubungan dengan struktur database dan koneksi API baik dari pihak internal maupun pihak ketiga. Saya sadar masih harus banyak belajar karena saya yakin proses pembelajaran itu adalah proses tanpa akhir dan selalu ada tantangan dalam perjalanannya.
